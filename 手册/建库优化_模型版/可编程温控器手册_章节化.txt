# 程序日程
程序日程 您可每天设置四个时间段，工作日和周末可设置不同参数。建议使用预设设置（如下表所示），可有效降低供暖/制冷能耗。 起床——设置起床时间及早晨离家前的所需温度。 离家——设置离家时间及外出时的所需温度（通常为节能温度）。 回家——设置回家时间及晚上就寝前的所需温度。 睡眠——设置就寝时间及夜间所需温度（通常为节能温度）。 <PIC>[Manual36_16]

# 智能响应技术
智能响应技术 该功能可让温控器“学习”炉子和空调达到程序设定温度所需的时间，确保在您设定的时间达到目标温度。例如：将起床时间设为早上6点，温度设为70°F。供暖系统会在6点前启动，确保您6点起床时温度已达到70°F。系统在预定时间段前启动时会显示“恢复中”提示 #

# 更换电池
更换电池 温控器需电池供电。电池低电量图标出现时请立即更换新电池。该图标会在电池耗尽前约两个月显示。 即使未出现电池低电量图标，也应每年更换一次电池，或离家超过一个月前更换。 若两分钟内完成电池更换，无需重新设置时间和日期。所有其他设置将永久存储在内存中，无需电池供电。 <PIC>[Manual36_26] 注：更换电池时建议使用碱性电池。 # 设置时间 1 按菜单键进入设备菜单 显示PROG。按+或-选择时间。 2 按选择键。 3 显示12小时制。按+或-选择12小时制或24小时制时钟。按选择键 4 设置时间。按+或-调整小时。按选择键。 5 按+或-调整分钟。按选择键。按选择键保存并退出。 <PIC>[thermostat_02]

# 故障排除、警告：触电风险、警告：汞处理...
故障排除 若温控器出现问题，请尝试以下建议 大多数问题均可快速简便解决。 # 屏幕空白 无法将系统设置切换至制冷模式 若使用电池供电，请确保已正确安装全新AA碱性电池（见第6页）。检查系统设置选项220，确保设置为1或2（见第9页）。 需要供暖时风扇不启动 检查系统设置选项205，确保设置与您的供暖设备匹配（见第8页）。 制冷模式下供暖系统运行 检查系统设置选项200或218，确保设置与您的供暖和制冷设备匹配（见第9页） # 供暖或制冷系统无响应 按系统键将系统设置为供暖。确保温度设定高于室内温度。 设置低于室内温度。检查断路器并在必要时复位。确保供暖和制冷系统的电源开关已打开。确保炉门紧闭 等待5分钟让系统响应。 # 屏幕上供暖开启/制冷开启闪烁 # 热泵问题 供暖模式吹冷风，或制冷模式吹热风 压缩机保护功能已启动。等待5分钟让系统安全重启，避免损坏压缩机。 检查系统设置选项200或218，确保设置与您的供暖和制冷设备匹配（见第9页）。 #
