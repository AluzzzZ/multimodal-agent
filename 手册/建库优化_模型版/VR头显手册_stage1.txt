# 注意事项
关于产品安全、使用、儿童、佩戴、屏幕、拆卸、结露、异常操作、高温、电源及存放的综合注意事项。

# 安全
禁止损坏电源线及电源适配器。

# 使用与操作
涵盖接口清洁、线缆甩动、追踪灯熄灭后操作、物理撞击、儿童接触、雷雨天气、液体异物进入、灰尘烟雾、长时间接触发热、皮肤不适、废弃处理、外壳老化、音量与听力保护、静电触击等操作要求。

# 儿童使用
VR头显不适用于12岁以下儿童。

# 佩戴VR头显
正确佩戴的重要性，包括调节旋钮勿过紧、可佩戴眼镜、定期检查佩戴状态。

# VR头显屏幕
屏幕可能出现暗点或常亮像素属正常现象，保持附着传感器无遮挡以防屏幕残影。

# 严禁拆卸或改装本产品及配件
禁止未经授权的拆解或改装，否则保修失效并存在风险。

# 结露
从寒冷环境移至温暖环境可能结露，需关闭电源待水分蒸发后再使用。

# 异常或无响应操作
长按电源键强制关机重启，若问题依旧则重启系统。

# 高温提示
处理器单元过热时屏幕显示提示，需关闭系统冷却并移至通风处，勿遮挡通风口。

# 电源适配器及电源线使用
使用易于插拔的电源，异常时立即断电，仅使用标配适配器，湿手勿插拔，勿覆盖适配器，清洁或长期不用时拔线，勿连接变压器或逆变器。

# 存放条件
避免高温、高湿、阳光直射，勿用烘干设备烘烤，勿放置在不稳定平面。

# 使用PS VR
PS VR需配合主机与摄像头使用，提供沉浸式体验，系统软件需更新至最新版本。

# 游玩区域
在摄像头捕捉范围内设置游玩区域，清除障碍物，按指示调整摄像头位置与角度，超出区域时屏幕警告。

# 使用头显时的姿势
尽可能保持坐姿，站立模式须严格遵循应用程序指示。
